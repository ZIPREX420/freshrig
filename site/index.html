Frequently Asked
Questions

What does the founder's lifetime include?
Lifetime updates to the desktop app, every future major version, no subscription, for the first 500 customers only. Excludes Encrypted Profile Sync (cloud infrastructure).
What happens if I cancel my subscription?
App keeps working in Free tier mode. Pro features lock until you resume. Your data stays on your device.
Can I pause my subscription?
Yes. Built-in pause option preserves your seat without billing.
Monthly vs annual?
Annual saves 32% (4 months free). Both have a 7-day free trial — no credit card required to start.
How does the multi-platform license work?
One license = unlimited devices on Windows + Linux + macOS for one user.
Can I switch from Pro to Pro Business?
Yes, prorated. Email support@freshrig.app.
Is my data private?
Absolutely. FreshRig reads your hardware info locally and never sends it anywhere. The only network requests are to GitHub (for auto-updates), to winget/apt/dnf/Homebrew servers (for app downloads), and to LemonSqueezy (for license validation). No analytics, no tracking, no telemetry.
Will it install bloatware or adware?
Never. FreshRig uses your platform's official package manager — winget on Windows, apt/dnf/pacman on Linux, Homebrew on macOS. Every app comes from verified sources with no bundled extras.
Will Windows SmartScreen warn me?
Yes, because FreshRig is a new app without a code signing certificate yet. Click "More info" → "Run anyway". The app is open source — you can inspect every line of code on GitHub.
