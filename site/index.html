Frequently Asked
Questions

Is FreshRig free?
The core app is free forever and open source under the MIT license — that includes hardware detection, drivers, app install, profiles, Windows optimization, and startup management. A Pro tier ($39 one-time, no subscription) unlocks six advanced features: Disk Cleanup, Privacy Dashboard, Network Tools, Services Manager, Context Menu Editor, and the System Health Report. Every install gets a 14-day free trial of Pro.
What does Pro cost and how many PCs can I use it on?
Pro is $39 one-time — no subscription, no renewal. One license activates on up to 3 PCs, which covers reinstalls and backup machines. Updates are free for life. Payment goes through LemonSqueezy (all major cards, Apple Pay, Google Pay).
Is my data private?
Absolutely. FreshRig reads your hardware info locally and never sends it anywhere. The only network requests are to GitHub (for auto-updates) and to winget servers (for app downloads). No analytics, no tracking, no telemetry.
Will it install bloatware or adware?
Never. FreshRig uses your platform's official package manager — winget on Windows, apt/dnf/pacman on Linux, Homebrew on macOS. Every app comes from verified sources with no bundled extras.
Does it work offline?
The hardware scan and profiles work offline. App installation requires an internet connection.
Will Windows SmartScreen warn me?
Yes, because FreshRig is a new app without a code signing certificate yet. Click "More info" → "Run anyway". The app is open source — you can inspect every line of code on GitHub.
How is this different from Ninite?
Ninite installs apps. FreshRig does that AND detects your hardware, recommends drivers, lets you save/share setup profiles, and shows your system health. It's the complete post-install toolkit.
Does FreshRig work on Linux?
Yes — since v1.1.0, FreshRig runs natively on Ubuntu 22.04+, Fedora 40+, Arch Linux, Linux Mint, Pop!_OS, openSUSE, and any distro with WebKitGTK 4.1. Downloads are available as .deb, .rpm, and .AppImage. Apps install via your distro's native package manager (apt, dnf, pacman, zypper) or Flatpak as a fallback.
Does FreshRig work on macOS?
Yes — since v1.2.0, FreshRig runs natively on macOS 11 (Big Sur) and later, including the latest Tahoe release. The download is a single universal .dmg that runs on both Apple Silicon (M-series) and Intel Macs. Apps install via Homebrew (casks for GUI apps, formulae for CLI tools). The first launch may show a Gatekeeper warning until our Apple Developer signing certificate is configured — right-click → Open to bypass it once.
